How To Rotate An Element in CSS


First we define the animation-name to use with our @keyframes (changeColor). The @keyframes controls the intermediate steps in our CSS animation sequence. It defines the waypoints for the keyframes along the animation sequence. Next we define the length of how long it takes for our animation to run through one sequence with animation-duration. This means in the case above it will take ten seconds to run one cycle aka for the three @keyframes to get from 0% to 100%. The last piece is the animation-iteration-count. This is stating how many iterations of the sequence we want to happen for one cycle.

Now let's look at the keyframes. In the example above, we have set our @keyframes to start in focus with the color black. By the 50% marker we have told our animation we want the color of our text to change as well as blur by 2px. And at the 100% marker or 10s, we will return back to the original state.
