BLOG

I decided after much debate to just start over. I kicked my Wordpress and purchased a Linux server. I wanted to build a website from nothing. Hard code it from scratch. My journey in music production has been nothing short of "I'll figure it out as I go". Programming has a lot of parallels in that regard. Somewhere along the way I came up with the idea for Chordo.

Which lead me to learning Python, then Flask, DNS, WSGI... It's like I knew what I was getting into...but was also forcing a bit of naivety. I started to write about all of this because I love writing, so why not? Programming is a lot like music. You start creating something and then you break it down. There's harmony in one's code. Also like music it's a whole lot of back and forthing. Back and forth back and forth -- forever in search for the Holy Grail.
