Deploying Your First LAMP Stack

03.09.24

Congratulations! You are here! Which means you are interested in building your first LAMP stack. “LAMP” is an acronym that represents the four main components of the stack:

  • 1. Linux: The Operating System
  • 2. Apache: The Web Server
  • 3. MySQL: The Database Server
  • 4. PHP: The Programming Language
All four of these are open source, which means they are community maintained and freely available for anyone to use. LAMP stacks are used to create, host, and maintain web content. Many of the websites we still use everyday are powered in this fashion. Where to build is the next question.

I chose IONOS as my server host. It’s cost effective and they have phenomenal support. Another option is AWS(Amazon Cloud). They offering a free-tier for the first year but with no support. So if you feel confident enough that’s a great option otherwise I’d suggest IONOS.

  • 1. Acquire Your Server / Choose Your Linux Distribution:
    Linux has a wide variety of distributions but for this tutorial we are going to use Ubuntu.

  • 2. SSH into your server:
    SSH or Secure Shell is a network communication protocol that enables two computers to communicate via the terminal. To learn more about SSH click here.

  • 3. Install Apache:
    Use the package manager on your Linux distribution to install the Apache web server.

    sudo apt-get update
    sudo apt-get install apache2

  • 4. Install MySQL:
    Install the MySQL database server. Use the package manager to install it on your Linux distribution.

    sudo apt-get install mysql-server

  • 5. Secure MySQL:
    Setting up the root password and security for your database

    sudo mysql_secure_installation

  • 6. Install PHP:
    Setting up the root password and security for your database

    sudo apt-get install php libapache2-mod-php php-mysql

  • 6. Reconfigure Apache for PHP:
    Reconfigure Apache to recognize PHP files

    sudo systemctl restart apache2

Okay now for the best part. In your terminal you are going to want to navigate to the folder /var/www/html. Here we are going to create our index.html file. Choose whichever text editor you feel comfortable with, personally I enjoy Vim. Paste the following into the document:


<!DOCTYPE html>
<html>
<body>
<h1>The Heading tag </h1>
<p>A Paragraph.</p>
<p>Happy Learning</p>
<p>Hello World! What's Crappenin'?</p>
</body>
</html>

Save and exit the text editor. Your server already has been assigned an IP address. Locate it and plug it into the address bar of your browser. http://your.ip.add.ress - Click enter and the text written into your document will now be displayed in you browser window.


Now start building out that website you've always dreamed of!!!
